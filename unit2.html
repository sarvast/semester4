Computer Graphics & Animation

Scan Converting Lines

  • Definition: The process of converting geometric primitives (like lines) into pixel coordinates on a raster grid is called scan conversion or rasterization.
  • Digital Differential Analyzer (DDA) Algorithm:
    • A line drawing algorithm based on the equation $y = mx + c$.
    • It calculates the next point by adding a step value ($\Delta x$ or $\Delta y$) to the current coordinate.
    • Disadvantage: It involves floating-point calculations (rounding off), which makes it slower than integer-based algorithms.
  • Bresenham’s Line Algorithm:
    • An accurate and efficient raster line-generating algorithm that uses only integer addition and subtraction.
    • It uses a Decision Parameter ($p_k$) to determine whether to choose the next pixel as $(x_k+1, y_k)$ or $(x_k+1, y_k+1)$.
    • If $p_k < 0$, the next point is $(x_{k+1}, y_k)$; otherwise, it is $(x_{k+1}, y_{k+1})$ (for slope $0 < m < 1$).

Scan Converting Circles & Clipping

  • Symmetry: A circle is highly symmetric (8-way symmetry). If you calculate one point $(x, y)$, you can determine 7 other points.
  • Midpoint Circle Algorithm: Uses a decision parameter to find the closest pixel to the true circle boundary. It avoids square root calculations.
  • Clipping: Definition: The process of determining which parts of an object are visible (inside a window) and removing the parts that are outside.
  • Cohen-Sutherland Line Clipping: Uses a 4-bit region code (TBRL) to identify position relative to the window.

Database Management System (DBMS)

E-R Modeling (Entity-Relationship Model)

  • Entity: A real-world object (animate or inanimate) that is easily identifiable (e.g., Student, Employee). Represented by a Rectangle.
  • Attributes: Properties that describe an entity (e.g., Name, Age). Represented by an Ellipse.
    • Simple: Atomic values (e.g., Roll No).
    • Composite: Subparts (e.g., Name $\rightarrow$ First, Last).
    • Multivalued: Multiple values (e.g., Phone). Represented by Double Ovals.
  • Relationship: An association between two or more entities. Represented by a Diamond.
  • Keys:
    • Primary Key: Uniquely identifies an entity. Underlined.
    • Weak Entity: Cannot exist without a strong owner entity. No primary key. Double Rectangle.

Software Engineering

Requirements Analysis

  • Requirement Engineering Process: Feasibility Study, Elicitation, Analysis, and Validation.
  • SRS (Software Requirement Specification): A document describing what the system should do.
    • Characteristics: Correctness, Unambiguous, Consistent, Verifiable.
  • Data Flow Diagrams (DFD): Used to model the flow of data.
    • Level-0 DFD (Context Diagram): Represents the entire system as a single bubble interacting with external entities.

Optimization Techniques

Game Theory Basics

  • Zero-Sum Game: One player's gain is exactly equal to the other player's loss.
  • Pure Strategy: Players select the same strategy every time. Exists if Maximin = Minimax.
    • Maximin: Max of row minimums (Player A).
    • Minimax: Min of column maximums (Player B).
  • Saddle Point: Where Maximin = Minimax. Value of the Game.
  • Dominance Rule: Used to reduce matrix size.
    • Row Dominance: If $R_i \le R_j$, delete $R_i$.
    • Column Dominance: If $C_i \ge C_j$, delete $C_i$.

Mathematics-III

Vector Calculus

  • Gradient ($\nabla \phi$): Acts on a Scalar function $\phi$. Result is a Vector. Represents normal to surface.
    $\nabla \phi = \frac{\partial \phi}{\partial x}\hat{i} + \frac{\partial \phi}{\partial y}\hat{j} + \frac{\partial \phi}{\partial z}\hat{k}$.
  • Divergence ($\nabla \cdot \vec{F}$): Dot product of $\nabla$ and Vector $\vec{F}$. Result is Scalar.
    If $\text{div } \vec{F} = 0$, vector is Solenoidal.
  • Curl ($\nabla \times \vec{F}$): Cross product. Result is Vector.
    If $\text{curl } \vec{F} = 0$, vector is Irrotational.

Which algorithm is considered a faster method for calculating pixel positions for line drawing?

A. DDA Algorithm
B. Bresenham’s Line Algorithm
C. Midpoint Algorithm
D. Parallel Line Algorithm
Bresenham's algorithm uses only integer arithmetic (addition/subtraction), making it computationally faster than DDA which uses floating-point values.

In the Cohen-Sutherland line clipping algorithm, the region code '1001' implies the point is in which region?

A. Top-Left
B. Top-Right
C. Bottom-Left
D. Bottom-Right
The 4-bit code is T-B-R-L. '1001' means Top (1) and Left (1) bits are set.

In an E-R diagram, a "Diamond" symbol represents:

A. Entity
B. Attribute
C. Relationship
D. Weak Entity
Relationships between entity sets are represented by a diamond-shaped box in E-R diagrams.

Which of the following represents a multivalued attribute in an E-R model?

A. Dashed Ellipse
B. Double Ellipse
C. Rectangle
D. Diamond
Multivalued attributes (like phone numbers) are depicted using double ovals (ellipses).

Which of the following is NOT a characteristic of a good SRS?

A. Consistent
B. Unambiguous
C. Brief
D. Verifiable
An SRS must be complete and detailed, not necessarily "brief". Briefness can lead to ambiguity.

The "Context Diagram" is also known as:

A. Level-0 DFD
B. Level-1 DFD
C. E-R Diagram
D. Use Case
Level-0 DFD represents the entire system as a single process interacting with external entities.

A game is said to be "fair" if the value of the game is:

A. 1
B. 0
C. Positive
D. Negative
If the value of the game is zero, strictly neither player has an advantage.

The condition for the existence of a Saddle Point is:

A. Maximin = Minimax
B. Maximin > Minimax
C. Maximin < Minimax
D. Maximin + Minimax = 0
A saddle point exists in a pure strategy game when the maximum of row minimums equals the minimum of column maximums.

If $\nabla \cdot \vec{F} = 0$, then the vector $\vec{F}$ is called:

A. Solenoidal
B. Irrotational
C. Rotational
D. Conservative
A vector whose divergence is zero is known as a Solenoidal vector.

The Gradient of a scalar function $\phi$ results in a:

A. Scalar Quantity
B. Vector Quantity
C. Constant
D. Zero
Gradient ($\nabla \phi$) introduces direction components, resulting in a vector.
