Topic-wise Notes

Computer Graphics & Animation

Geometrical Transformations (2D)

  • Concept: Methods to modify the shape, size, or position of an object.
  • Translation: Moving an object. Equations: $x' = x + t_x$, $y' = y + t_y$.
  • Rotation: Rotating by angle $\theta$. Positive $\theta$ is Counter-Clockwise (CCW).
    • Equation: $x' = x \cos\theta - y \sin\theta$, $y' = x \sin\theta + y \cos\theta$.
  • Scaling: Changing size coordinates. $x' = x \cdot s_x$, $y' = y \cdot s_y$.
  • Reflection: Mirror image about an axis ($x$-axis, $y$-axis, $y=x$).
  • Shearing: Slanting the object. X-shear: $x' = x + sh_x \cdot y, y' = y$.
  • Homogeneous Coordinates: Representing 2D points as $(x, y, 1)$ to allow matrix multiplication for translation.

Database Management System (DBMS)

Data Normalization

  • Definition: Organizing data to minimize redundancy and dependency. Decomposes large tables.
  • 1NF (First Normal Form): Eliminates repeating groups. Attributes must be atomic.
  • 2NF (Second Normal Form): 1NF + No Partial Dependency (Non-key attributes depend on the full primary key).
  • 3NF (Third Normal Form): 2NF + No Transitive Dependency (Non-key attributes depend only on the primary key, not other non-keys).
  • BCNF (Boyce-Codd Normal Form): Stronger 3NF. For functional dependency $X \to Y$, $X$ must be a super key.

Software Engineering

Software Design Concepts

  • Software Design: Transforming SRS into a structure for implementation.
  • Modularity: Dividing software into distinct, addressable components.
  • Coupling: Interdependence between modules. Goal: Low Coupling.
    • Worst: Content Coupling. Best: Data Coupling.
  • Cohesion: Strength of relationship within a module. Goal: High Cohesion.
    • Best: Functional Cohesion. Worst: Coincidental Cohesion.
  • Structure Chart: Graphical representation of system modules and hierarchy.

Optimization Techniques

Replacement Theory

  • Replacement Problem: Deciding when to replace equipment due to deterioration or failure.
  • Gradual Failure: Efficiency drops over time (Machines, Cars). Replace when Maintenance Cost > Average Cost.
  • Sudden Failure: Works perfectly until it fails (Light bulbs).
  • Policies:
    • Individual: Replace on failure.
    • Group: Replace all at fixed time $T$. Good for low-cost bulk items.

Mathematics-III

Fourier Series

  • Definition: Expression of a periodic function $f(x)$ as a sum of sines and cosines.
  • Formula: $f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos(nx) + \sum_{n=1}^{\infty} b_n \sin(nx)$.
  • Euler's Formulas:
    • $a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) dx$
    • $a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos(nx) dx$
    • $b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin(nx) dx$
  • Even Function ($f(-x) = f(x)$): Symmetric about Y-axis. $b_n = 0$.
  • Odd Function ($f(-x) = -f(x)$): Symmetric about Origin. $a_0 = 0, a_n = 0$.
