Topic-wise Notes

Computer Graphics & Animation

Computer Animation

  • Definition: Creating fluid motion by displaying a sequence of still images (frames) quickly.
  • Frame Rate: Smooth motion typically requires 24-30 fps.
  • Key Concepts:
    • Keyframing: Defining start and end states; computer interpolates intermediates.
    • Tweening (In-betweening): Generating intermediate frames between keyframes.
  • Morphing: Transforming one image into another. Steps: Warping (Shape distortion) & Cross-dissolving (Color fading).
  • Motion Specifications:
    • Direct Motion: Explicitly setting rotation/translation values.
    • Goal-Directed: Defining the final state (e.g., "Walk to X").
  • 3D Animation Process: Modeling $\to$ Layout/Animation $\to$ Rendering.

Database Management System (DBMS)

Transaction Processing & Compulsory Control

  • ACID Properties:
    • Atomicity: All or nothing.
    • Consistency: DB remains valid.
    • Isolation: Transactions don't interfere.
    • Durability: Saved permanently.
  • Concurrency Control: Managing simultaneous transactions.
    • Locking: Shared (Read) vs. Exclusive (Write).
    • 2PL (Two-Phase Locking): Ensures Serializability. Phases: Growing (Lock) & Shrinking (Unlock).
    • Deadlock: Two transactions waiting for each other indefinitely.
  • Recovery:
    • Log-Based (WAL): Recording changes before writing to disk.
    • Checkpoints: Saving logs to disk to **reduce recovery time**.
    • Shadow Paging: Using two page tables for atomic recovery.

Software Engineering

Software Maintenance

  • Definition: Modifying software after delivery.
  • Types of Maintenance:
    1. Corrective: Fixing bugs/errors (e.g., login failure).
    2. Adaptive: Adjusting to new environments (OS/Hardware changes).
    3. Perfective: Adding features/improving performance (User requests).
    4. Preventive: Refactoring to prevent future issues.
  • Reverse Engineering: Analyzing a system to identify components and create a higher-level representation.

Optimization Techniques

Job Sequencing

  • Goal: Determine order of jobs on machines to minimize **Total Elapsed Time**.
  • Johnson's Rule (n Jobs, 2 Machines):
    1. Find minimum processing time in the list.
    2. If on Machine 1 $\to$ Process First.
    3. If on Machine 2 $\to$ Process Last.
    4. Repeat until all jobs sequenced.
  • Idle Time: Time a machine is waiting.

Mathematics-III

ODE of Second Order

  • General Form: $\frac{d^2y}{dx^2} + P\frac{dy}{dx} + Qy = R(x)$.
  • Solution ($y$): Complementary Function (C.F.) + Particular Integral (P.I.).
  • Finding C.F. (Auxiliary Equation $m^2+Pm+Q=0$):
    • Real & Distinct ($m_1, m_2$): $y = c_1 e^{m_1x} + c_2 e^{m_2x}$.
    • Real & Equal ($m, m$): $y = (c_1 + c_2x)e^{mx}$.
    • Complex ($\alpha \pm i\beta$): $y = e^{\alpha x}(c_1 \cos \beta x + c_2 \sin \beta x)$.
  • Finding P.I.:
    • If $R(x) = e^{ax}$, P.I. = $\frac{e^{ax}}{f(a)}$ (provided $f(a) \neq 0$).
    • Case of Failure: Multiply by $x$ and differentiate denominator.
